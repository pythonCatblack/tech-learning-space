IMU（Inertial Measurement Unit，惯性测量单元）是飞控系统中最重要的单一传感器。它由加速度计（测三轴加速度）和陀螺仪（测三轴角速度）组成，以最高频率持续采样，直接测量机体相对于惯性空间的运动状态。IMU的核心价值在于：极高频率、极低延迟、全自由度——但代价是误差会随时间快速累积（漂移）。理解IMU的物理和误差特性，是设计传感器融合系统的第一步。

5.1 IMU的工作原理：MEMS微机电系统

5.1.1 机械弹簧与传感电容

MEMS IMU的核心结构是微型机械振子——一个在硅片上刻蚀出来的微型弹簧质量块系统。当机体加速时，质量块因惯性保持相对静止，与外壳产生相对位移；这个位移改变固定电极与质量块之间的电容，通过测量电容变化推算出加速度。

🔑 MEMS加速度计工作原理

质量块通过弹簧悬挂在硅腔内。沿某一轴加速时，质量块因惯性移动距离 d，电容变化 ΔC 与 d 成正比，通过电路转换为电压输出：

V_out ∝ ΔC ∝ a（加速度）

陀螺仪的原理类似但更复杂：利用科里奥利力（Coriolis Force）——当质量块在某个方向振动时，旋转运动会改变振动方向，通过检测振动方向的改变推算角速度。

物理直觉：想象你坐在旋转木马上，向前推动一个摆锤。旋转产生的科里奥利力会让摆锤向侧面偏移——这个力与旋转角速度成正比。MEMS陀螺仪就是用微型"摆锤"来测量角速度。

5.1.2 IMU的关键参数

📏

加速度计量程

常见量程：±2g（无人机悬停）、±8g（高速飞行）、±16g或±40g（特殊情况）。超出量程会饱和，产生非线性误差。

🔄

角速度量程

常见量程：±250°/s（常规姿态）、±1000°/s（特技飞行）、±2000°/s（高速无人机）。

📈

采样率

飞控IMU通常1-8kHz，数据输出率通常1kHz。采样率越高，积分精度越高，但计算负担也越大。

🧊

零偏稳定性

衡量陀螺仪漂移的关键指标，单位°/h（度/小时）。军用光纤陀螺仪<0.001°/h，MEMS IMU通常1-100°/h。

5.2 惯性导航方程

5.2.1 连续时间积分：速度与位置的计算

IMU测量的比力（specific force）需要经过积分才能得到速度和位置。比力是加速度计读数减去重力加速度后的值。

🔑 惯性导航基本方程

设机体在惯性空间的速度为 v，加速度为 a，重力加速度为 g（方向向下，约为9.81 m/s²），则：

dv/dt = a_measured - g（加速度积分得到速度）
dp/dt = v（速度积分得到位置）

从位置 p(0) 和速度 v(0) 的初始值出发，对加速度积分一次得到速度，积分两次得到位置。

离散化形式（欧拉积分）：
vₖ = vₖ₋₁ + (aₖ₋₁ - g) · Δt
pₖ = pₖ₋₁ + vₖ₋₁ · Δt + 0.5 · (aₖ₋₁ - g) · Δt²

5.2.2 为什么积分会累积误差？

关键问题：加速度测量存在偏差。假设加速度偏差为 δa（哪怕只有 0.01 m/s²），积分两次后：

误差累积的数学

积分两次，误差变成二次方增长

• t秒后速度误差 = δa · t
• t秒后位置误差 = ½ · δa · t²

举例：加速度偏差 δa = 0.01 m/s²（0.1%的量程误差）
• 1秒后：速度误差 0.01 m/s，位置误差 0.005m（5mm）
• 10秒后：速度误差 0.1 m/s，位置误差 0.5m（半米）
• 60秒后：位置误差达到 18m！

这就是为什么纯惯性导航在60秒内就会产生十几米的定位误差——IMU只能作为短时参考，必须融合其他传感器（GPS/视觉/Lidar）来修正累积误差。这也是第9章卡尔曼滤波和第10章因子图优化的核心动机。

5.3 IMU误差模型

5.3.1 确定性误差

⚖️

零偏 (Bias)

加速度计或陀螺仪在零输入时的输出偏移。常温下通常稳定，高低温切换时会有变化（零偏温漂）。消费级MEMS的零偏通常在 ±5-100 mg / ±1-10 °/s。

📐

比例因子误差 (Scale Factor)

输出与真实值之间的比例系数偏差。通常以ppm（百万分之一）为单位。MEMS比例因子误差约 ±500-2000 ppm。

🔀

轴对齐误差 (Misalignment)

三轴加速度计和陀螺仪的敏感轴不完全正交，存在微小角度偏差。通常 < 0.1°。

5.3.2 随机性误差：艾伦方差（Allan Variance）

IMU随机误差的频域特性通常用艾伦方差（Allan Variance）来分析。艾伦方差图揭示了IMU噪声的不同来源——这是选型IMU和配置滤波器Q/R矩阵的必读数据表。

📉

角度随机游走 (ARW)

噪声密度与 1/√τ 成正比。来源于白噪声积分，高频噪声主导。陀螺仪典型值：0.1-5°/√h。

物理含义：如果你的陀螺仪ARW=0.3°/√h，意味着飞行1小时后，仅由角度随机游走导致的角度误差标准差约为0.3°。

🔄

零偏不稳定性 (Bias Instability)

低频噪声，与 1/τ 相关。代表陀螺仪长期漂移特性。陀螺仪典型值：1-100°/h。

物理含义：如果你的陀螺仪零偏不稳定性=10°/h，意味着即使在静止条件下，陀螺仪的零偏也会在小时内漂移10°。

🔑 如何解读艾伦方差图

艾伦方差图的读取方法（以典型MEMS陀螺仪为例）：

斜率 -0.5（高频区）：对应角度随机游走（ARW），曲线的截距即ARW系数。
斜率 +0.5（中频区）：对应零偏不稳定性，曲线最低点即零偏不稳定性值。
斜率 +1（低频区）：对应随机游走偏差（Walk Random）。

实战应用：从艾伦方差图可以读出每个噪声源的幅度，直接用于配置Kalman滤波器的Q矩阵（过程噪声协方差）。例如：ARW=0.3°/√h 意味着在1秒的积分中，角度误差的标准差 = 0.3°/√h × √1h = 0.3°（仅1秒就产生0.3°的角度误差！）。这就是为什么VIO不能纯靠IMU积分。

5.3.3 IMU随机游走误差的数学表示

在传感器融合算法中，IMU的连续时间随机误差模型通常表示为：

🔑 IMU随机误差模型

陀螺仪随机游走（角度随机游走）：
ω(t) = ω_true(t) + b(t) + n(t)
其中 b(t) 是随机游走过程（ḃ = n_b，n_b 是高斯白噪声），n(t) 是角度随机游走（白噪声）。

加速度计：
a(t) = a_true(t) + b_acc(t) + n_acc(t)

这个模型直接决定了 Kalman滤波中IMU噪声协方差矩阵的设置——如果你不理解这个模型，就不知道如何正确配置滤波器的噪声参数。实践中，很多VIO系统的定位精度差，根源就是Q/R矩阵配置不当（见第9章9.4节）。

5.4 机体坐标系与导航坐标系

5.4.1 坐标系定义

飞控系统涉及多个坐标系，清晰地区分它们至关重要：

✈️

机体坐标系 (Body Frame)

原点位于无人机重心，X轴指向前方，Y轴指向右方，Z轴指向下方（飞控行业常用）。相对于导航坐标系旋转。

🌍

导航坐标系 (Nav Frame)

通常取当地北-东-地（NED）或东-北-天（ENU）坐标系。绝对参考系，位置和速度在此坐标系下定义。

💨

IMU坐标系

IMU芯片的本地坐标系。由于制造公差，IMU坐标轴与机体坐标轴存在微小偏差（misalignment），需要标定。

5.4.2 姿态表示：欧拉角、四元数与旋转矩阵

描述机体相对于导航坐标系的姿态（方向），有三种等价的数学表示：

🔄

欧拉角

(Roll, Pitch, Yaw) 最直观，人类可读。但存在万向锁（Gimbal Lock）问题，不适合插值和滤波。

🎯

四元数 (Quaternion)

q = [q₀, q₁, q₂, q₃]，四个参数，无万向锁，适合插值和滤波。飞控融合算法中的标准姿态表示。

📊

旋转矩阵 (Rotation Matrix)

3×3正交矩阵 R，满足 RRᵀ = I，det(R) = 1。冗余度最高（9个参数），但计算最直接。

🔑 四元数基础运算

四元数乘法（用于合成旋转）：
设 q = [q₀, qᵥ], p = [p₀, pᵥ]，其中 qᵥ 和 pᵥ 是向量部分。
q ⊗ p = [q₀p₀ - qᵥ·pᵥ, q₀pᵥ + p₀qᵥ + qᵥ×pᵥ]

旋转矩阵转四元数：
q₀ = 0.5 × √(1 + tr(R))
q₁ = (R₃₂ - R₂₃) / (4q₀)
q₂ = (R₁₃ - R₃₁) / (4q₀)
q₃ = (R₂₁ - R₁₂) / (4q₀)

用四元数旋转向量：
v' = q ⊗ [0, v] ⊗ q⁻¹
其中 q⁻¹ = [q₀, -qᵥ] / ||q||（共轭除以模长）

⚠️
四元数是飞控融合的核心：所有主流飞控（PX4、ArduPilot）和VIO系统（OKVIS、VINS-Mono）内部都用四元数表示姿态。四元数的优势：存储效率高（4个浮点数）、插值平滑、不存在万向锁、乘法运算高效。务必理解四元数的乘法规则和旋转变换——这是第9章EKF和第10章因子图的基础数学工具。
▼📐 深度：艾伦方差图的正确解读方法

艾伦方差（Allan Variance, AVAR）是分析IMU随机噪声频谱特性的标准方法。本节详解如何从AVAR图中提取关键噪声参数。

艾伦方差的定义：

σ²(τ) = ½ · E[(O(t+τ) - O(t))²]

其中 O(t) 是传感器输出，τ 是簇持续时间。
对静止数据分组，取不同τ值计算方差，画在对数坐标上 → AVAR曲线。

AVAR曲线各段的物理含义：

斜率对应噪声源可读取的参数
-½角度随机游走（ARW）噪声密度 N（°/√h）
0零偏不稳定性B（°/h，最低谷值）
+½随机游走偏差K（°/h³/²）
+1速率斜坡R（°/h²）

实战解读步骤：

1. 在AVAR图上找到斜率=-½（高频段）的直线
2. 外推该直线到τ=1h，与AVAR曲线的交点 → ARW系数 N（单位：°/√h）
3. 找到曲线的最低点 → 零偏不稳定性 B（°/h）

示例：某MEMS陀螺仪的AVAR图：
  τ=1s时：σ(1s) ≈ 0.01°/s（角度随机游走主导）
  τ=1000s时：σ(1000s) ≈ 0.5°/h（零偏不稳定性主导）

使用：将N代入Kalman滤波Q矩阵：
  Q_gyro = N² · Δt （离散时间噪声方差）
本章关键概念

IMU是融合系统的"时间基准"

① MEMS IMU原理：利用惯性力和科里奥利力，通过电容变化测量加速度和角速度。
② 惯性导航方程：速度是加速度的时间积分，位置是速度的时间积分，积分两次后误差随时间二次方增长。
③ 艾伦方差：分析IMU噪声频谱特性的标准方法，揭示角度随机游走和零偏不稳定性。
④ 四元数：飞控融合系统的标准姿态表示，无万向锁、适合插值和优化。
⑤ IMU预积分：VIO系统中将高频IMU数据压缩为两帧间的相对运动约束，避免重复积分。
