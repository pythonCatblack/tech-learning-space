无人机的"精准触角"

🔗
前置依赖：Chapter 3（相机模型）帮助理解投影几何，本章的点云-图像联合处理思想是第12章 Lidar-视觉融合的基础。建议完全掌握相机内参和外参概念后再进入本章。

激光雷达（Lidar = Light Detection and Ranging）是目前最精确的距离传感器——在有效范围内，测距精度可达±2cm，远超相机（双目在远距离时误差可达米级）。它直接输出稠密的三维点云（Point Cloud），每个点包含精确的 XYZ 坐标和反射强度（Intensity）。但Lidar也有明显弱点：无法获取颜色/纹理信息、对恶劣天气敏感、成本较高。理解Lidar的物理特性，是设计鲁棒感知系统的前提。

4.1 激光雷达的物理原理

4.1.1 飞行时间法（ToF）：光速测量距离

最常见的Lidar测距原理是飞行时间法（Time of Flight，ToF）。工作流程：

1

发射激光脉冲

发射器向目标方向发射一束近红外激光脉冲（波长通常905nm或1550nm）。

2

光子飞行与反射

光以光速 c ≈ 3×10⁸ m/s 飞行，遇到目标表面后反射。

3

接收返回光子

接收器检测返回的光子，记录发射和返回的时间差 Δt。

4

计算距离

距离 R = c · Δt / 2（往返，除以2）。

🔑 测距精度与光速的关系

光速约 3×10⁸ m/s，意味着：
• 1纳秒（10⁻⁹s）对应距离约 15cm
• 1皮秒（10⁻¹²s）对应距离约 0.15mm

高端Lidar使用时间数字转换器（TDC）可以达到皮秒级精度，对应毫米级测距精度。但消费级Lidar通常只能达到厘米级——不是因为光速不够快，而是计时精度不够高。工程上，Livox Horizon的测距精度为 ±2cm@1σ（1 sigma）。

4.1.2 FMCW调频连续波法

FMCW（Frequency Modulated Continuous Wave）是另一种测距原理——不是测量光飞行时间，而是测量频率差。发射一束线性调频的激光，返回光与发射光产生干涉，测量频差即可得到距离。

📊

ToF优势

技术成熟、成本较低；峰值功率高，适合远距离探测。

📈

FMCW优势

可同时测速（多普勒效应）；抗干扰能力强（相干接收）；人眼安全功率更低（1550nm）。

💡
关键洞察 - FMCW的多普勒测速：FMCW激光雷达不仅可以测距，还能通过多普勒效应直接测量每个点的径向速度。这意味着一个点就能同时告诉你"距离多远"和"速度多少"——这对动态障碍物跟踪非常有价值。Mobileye的下一代车载Lidar采用FMCW正是基于此原因。对无人机而言，这项能力在高速飞行时的避障决策中至关重要。

4.2 激光雷达的类型

4.2.1 机械旋转式Lidar

机械旋转式Lidar通过电机带动整个光学扫描结构旋转，实现360°水平视场（FOV）。这是最成熟的Lidar形态。

🔄

Velodyne系列

64线/32线/16线，机械旋转，2010年代自动驾驶研究的标准配置。重量大（~1.2kg）、功耗高（60W+）、成本高（数万美元）。

🛢️

Ouster OS0/OS1/OS2

数字Lidar，采用CMOS SPAD探测器，128/64/32线可选，更紧凑可靠。

4.2.2 固态/半固态Lidar

固态Lidar（Solid-State Lidar）没有机械旋转部件，通过相控阵或微振镜实现扫描，可靠性更高、体积更小、成本有望大幅降低。

微振镜Lidar：用一个或两个微振镜实现水平+垂直扫描，如Livox Horizon 相控阵Lidar：用电信号控制光束方向，无机械移动部件（仍在研发阶段） Flash Lidar：一次照射整个视场，探测器同时接收，无扫描机制

4.3 点云数据结构与特性

4.3.1 点云的基本属性

Lidar输出的每个点（Point）包含以下属性：

📍

XYZ（坐标）

相对于Lidar坐标系的三维位置。通常以Lidar安装位置为原点，单位通常是米（m）。

💡

Intensity（反射强度）

返回光子的强度，与目标表面材质和入射角相关。金属/白色表面反射强，黑色/吸光表面反射弱（可能是安全服的反光条！）。

⏱️

Timestamp（时间戳）

每个点的时间戳，对于动态校正和与IMU同步至关重要。机械Lidar一个扫描周期内不同角度的点对应不同时间戳。

🌈

Line/Scan ID（某些型号）

多线Lidar中，每个点属于哪一条扫描线。

4.3.2 点云的特征与挑战

与图像相比，点云有独特的性质：

✅

优势

精确的XYZ坐标（直接3D几何）；主动光源，不受光照影响；不需标定（自带坐标）；每个点有时间戳。

❌

劣势

稀疏性（64线Lidar在100m处，每平方米可能只有几个点）；无颜色/纹理信息；动态物体上点云会有运动畸变（见4.5节）。

4.4 点云处理算法

4.4.1 点云滤波：去除噪声与下采样

原始点云通常包含噪声和离群点。常用滤波方法：

📉

体素滤波 (Voxel Filter)

将空间划分为等体积的体素网格，每个体素内只保留一个代表点。是最常用的点云下采样方法，保留几何结构的同时大幅减少点数。实战参数：体素大小通常选 0.1-0.5m，对于室内场景用0.05m，室外用0.2-1.0m。

🔄

统计离群点移除

对每个点的邻域做统计分析，距离邻域均值过远的点判定为离群点并移除。实战参数：K近邻数通常取20-50，标准差阈值取1.0-2.0（超过即判定为离群点）。

4.4.2 地面提取：无人机感知的"第一刀"

对于无人机避障和定位，最重要的一步是从点云中提取地面。地面将空间划分为"可飞行区域"和"障碍物区域"。

📐

基于高度差的方法

计算每个点与局部平面的高度差，超过阈值的点归类为障碍物。简单高效，但坡道场景会失效。

🔢

基于拟合的方法

用RANSAC或多项式拟合地面点云，再将剩余点分类为障碍物。对不平坦地形更鲁棒。

🗺️

基于栅格地图的方法

将空间划分为二维网格，每个格子记录最低点高度，高于地面阈值的点视为障碍物。广泛应用于无人机路径规划。

4.4.3 点云分割：从原始数据到语义物体

地面提取后，需要将剩余点云分割为独立物体（树干、电线杆、车辆、人）：

📊

传统分割

欧式聚类（基于距离）、区域生长法、法向量聚类。对规则场景效果好，计算快。

🤖

深度学习分割

PointNet++、PointPillars、SqueezeSeg——直接从原始点云预测语义类别。精度高但需要GPU实时推理。

4.5 运动畸变校正

机械旋转式Lidar在扫描一整圈（通常100ms-1s）期间，无人机本身在移动。这会导致点云产生运动畸变——同一条扫描线上，远离无人机的地方畸变更大。例如：无人机以5m/s向前飞行，100ms扫描周期内无人机前进了50cm——这意味着扫描起始点和结束点之间有50cm的位置差异，在高速飞行时这是不可忽略的。

🔑 运动畸变校正的核心思想

利用IMU或里程计提供的短时姿态信息，对每个点的扫描时刻做逆向变换：

步骤1：记录每个点的精确时间戳（来自Lidar数据包）
步骤2：用IMU数据插值出该时刻的无人机姿态
步骤3：将该点投影到扫描结束时刻的坐标系下

设扫描起始时刻为 t₀，结束时刻为 t₁，某点扫描时刻为 t，其对应的无人机姿态为 R(t)，平移为 p(t)。校正后的点坐标：
P_corrected = R(t₁) · R(t)⁻¹ · (P_raw - p(t) + p(t₁))

校正后的点云消除了运动模糊，几何精度显著提升。这是Lidar SLAM预处理的标准步骤，不做校正会导致建图精度下降和里程计漂移。

🛠️ 动手实践

用PCL可视化点云并实现体素滤波

目标：用Python (open3d或pypcl) 读取Lidar点云数据，可视化并实现体素滤波下采样。

# 安装 open3d
pip install open3d

import open3d as o3d
import numpy as np

# 读取点云（支持 PCD/PLY/TXT格式）
pcd = o3d.io.read_point_cloud("lidar_scan.pcd")
print(f"原始点数: {len(pcd.points)}")

# 体素滤波下采样
voxel_size = 0.2  # 20cm 体素大小
pcd_down = pcd.voxel_down_sample(voxel_size)
print(f"下采样后: {len(pcd_down.points)}")

# 统计离群点移除
cl, ind = pcd_down.remove_statistical_outlier(nb_neighbors=20, std_ratio=1.0)
pcd_clean = pcd_down.select_by_index(ind)

# 可视化
o3d.visualization.draw_geometries([pcd_clean])
进阶：实现地面提取——计算每个点与 Z=地面高度 的差值，低于阈值的点标记为地面点，剩余点为障碍物。
数据集推荐：下载 NCLT 数据集（密歇根大学自动驾驶数据集，含Velodyne 32线Lidar和IMU数据），用上面的代码读取一帧并可视化。

⚠️
常见陷阱① - 忽视运动畸变：很多初学者直接用原始点云做建图，发现轨迹在高速飞行时严重扭曲。必须先用IMU数据做运动畸变校正（4.5节），再送入LOAM/LIO-SAM。

常见陷阱② - Livox点云密度不均匀：Livox中心区域点密度远高于边缘，如果直接用于需要均匀点云的算法（如某些欧式聚类方法），效果会差很多。先用非均匀加权采样处理。

常见陷阱③ - 黑色吸光表面：深色衣服、深色车辆、树木在Lidar上几乎没有回波——它们不会出现在点云中，但你不能假设"没有点的地方就是空的"。这对安全至关重要：穿黑色衣服的人在Lidar眼里可能是"透明的"。

Lidar感知的技术栈
