2.1.2 文本转3D模型操作流程

1

步骤1：进入文本生成模式

点击左侧工具栏的"Text to 3D"按钮，进入文本生成模式。

2

步骤2：输入描述文本

在文本框中输入您想要生成的3D模型描述。建议使用英文描述以获得最佳效果。

3

步骤3：调整生成参数

设置模型风格（通用/卡通/写实）、分辨率、是否包含纹理等参数。

4

步骤4：生成与预览

点击"Generate"按钮，等待AI生成模型。生成完成后可以在画布区域旋转、缩放预览。

5

步骤5：导出模型

满意后点击"Export"按钮，选择导出格式（GLB/FBX/OBJ等）。

💡 提示词示例
推荐使用的英文描述格式：

基础描述：
"a cute robot toy with round head and big eyes"

详细描述：
"a vintage wooden bookshelf, 1 meter tall, with 4 shelves,
brown oak wood texture, classic design"

包含材质：
"a red ceramic coffee mug with white interior,
glossy finish, handle on the right side"

2.1.3 图片转3D模型操作流程

1

步骤1：进入图片生成模式

点击左侧工具栏的"Image to 3D"按钮。

2

步骤2：上传参考图片

点击上传按钮，选择一张清晰的产品图片或设计图。建议使用多角度图片（3-5张）以获得更好的生成效果。

3

步骤3：设置参数

选择生成模式（快速/精细）、是否自动补全背面等。

4

步骤4：生成与编辑

生成完成后可以对模型进行局部编辑，如调整特定区域的结构或材质。

2.3 开源模型实操：TripoSR

TripoSR是由Stability AI与VAST联合开发的开源模型，0.5秒即可从单张图片生成高质量3D模型，完全免费可商用。

2.3.1 TripoSR部署方式

☁️

云端体验

通过HyperAI、OpenBayes等平台在线体验，无需本地部署

💻

本地部署

从GitHub克隆代码，自行配置Python环境运行

2.3.2 云端使用流程（推荐）

1

访问在线Demo

访问HyperAI官网的TripoSR教程页面，或在OpenBayes上搜索"TripoSR"

2

克隆教程

点击"克隆"将教程复制到自己的容器中

3

上传图片

在Demo界面上传单张产品图片

4

生成与下载

等待约0.5-2秒生成完成，下载GLB/OBJ格式模型

2.3.3 本地部署命令

💻 TripoSR本地部署代码

git clone https://github.com/VAST-AI-Research/TripoSR.git
cd TripoSR

# 创建虚拟环境
conda create -n trisr python=3.10
conda activate trisr

# 安装依赖
pip install torch torchvision torchaudio
pip install -r requirements.txt

2.6 常用快捷键与操作技巧

💡 3D画布通用快捷键

- 数字键1：正视图

- 数字键7：俯视图
- 数字键0：等轴测图

2.7 模型精度控制与优化技巧

2.7.1 影响模型精度的因素

因素高精度效果低精度效果
多边形数量细节丰富，曲面平滑棱角分明，面状感明显
纹理分辨率细节清晰，质感真实模糊，像素化明显
UV展开质量贴图对齐准确，无拉伸贴图错位，变形明显

2.7.2 优化建议

✅

合理选择精度

屏幕展示用低精度即可；3D打印根据打印机精度调整

✅

减少不必要的细节

远处物体可用简化模型，近处物体才需要高精度

✅

使用LOD技术

同一物体准备多个精度版本，根据需要切换

2.8 常见问题诊断与解决

问题现象可能原因解决方案
生成失败网络连接问题/描述不符合物理规律检查网络，重试；简化描述词
模型变形输入图片质量低/物体被遮挡使用清晰无遮挡的图片
多边形过多AI生成精度过高使用减面工具优化
纹理错位UV展开问题重新烘焙或手动调整UV
导出格式不支持工具不支持该格式转换为支持的中间格式如OBJ
