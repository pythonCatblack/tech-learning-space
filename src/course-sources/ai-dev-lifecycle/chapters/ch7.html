AI代码生成可以分为几个层次：语法级生成、函数级生成、模块级生成、架构级生成。层次越高，AI的作用越大，但也需要更精准的Prompt和更多的审查。

7.1 代码生成的层次

语法级
如Copilot的实时补全，单行或短代码块
→
函数级
根据注释或函数签名生成完整函数
→
模块级
根据需求描述生成一个功能模块（多个文件）
→
架构级
AI辅助搭建项目结构，生成基础架构代码

7.2 框架代码的AI生成

使用Fastify + TypeScript创建REST API

💡 提示词示例
使用Fastify + TypeScript创建一个任务管理的REST API框架。

要求：
1. 使用Fastify 4.x版本
2. TypeScript strict模式
3. 实现以下endpoints：
   - GET /tasks - 获取任务列表（支持分页和筛选）
   - GET /tasks/:id - 获取单个任务
   - POST /tasks - 创建任务
   - PUT /tasks/:id - 更新任务
   - DELETE /tasks/:id - 删除任务
4. 使用JSON Schema做请求/响应验证
5. 集成Zod做运行时验证
6. 错误处理统一封装
7. 包含健康检查endpoint

请生成：
- src/index.ts（入口文件）
- src/routes/tasks.ts（任务路由）
- src/schemas/task.ts（Zod schema）
- src/services/task.service.ts（业务逻辑服务）
- src/types/task.ts（TypeScript类型）
- package.json依赖配置

7.3 业务逻辑的AI辅助

业务逻辑往往涉及复杂的领域知识，AI无法完全理解业务背景，但可以帮助实现模式化的逻辑。

示例：任务优先级计算

💡 提示词示例
为任务管理系统实现优先级计算逻辑。

业务规则：
1. 紧急程度（基于截止日期）：
   - 已过期：100分
   - 24小时内到期：80分
   - 3天内到期：60分
   - 7天内到期：40分
   - 7天后到期：20分

2. 重要程度（基于标签）：
   - 有'紧急'标签：+30分
   - 有'重要'标签：+20分
   - 有'客户'标签：+15分

3. 阻塞惩罚：
   - 被其他任务阻塞：-50分
   - 阻塞其他任务：+10分

4. 最低优先级为0，最高不超过100

请使用TypeScript实现这个优先级计算函数，要求：
- 纯函数，无副作用
- 完整的类型定义
- 单元测试覆盖主要场景
- 包含JSDoc注释

7.4 代码重构与优化

AI可以帮助识别代码异味（Code Smell）并提供重构建议。

💡 提示词示例
请审查以下代码，指出可能的问题并提供重构建议。

关注点：
1. SOLID原则违反情况
2. 可能的性能问题
3. 错误处理不完善的地方
4. 可测试性问题
5. 安全漏洞

原始代码：
```typescript
class TaskManager {
  private tasks: any[] = [];

  addTask(task: any) {
    if (!task.title) throw new Error('Title required');
    if (!task.dueDate) throw new Error('Due date required');
    this.tasks.push(task);
    this.notify();
  }

  getTasks() {
    return this.tasks;
  }

  private notify() {
    // 发送通知的逻辑
    console.log('Task added');
  }
}
```

```sql
SELECT t.*, u.name as assignee_name
FROM tasks t
LEFT JOIN users u ON t.assignee_id = u.id
WHERE t.project_id = 123
AND t.status IN ('pending', 'in_progress')
AND t.created_at > '2024-01-01'
ORDER BY t.priority DESC, t.due_date ASC;
```

7.7 本章小结

📚 核心要点
  1. AI代码生成可分为语法级、函数级、模块级、架构级四个层次
  2. 框架代码生成是AI最擅长的场景，能大幅提升开发效率
  3. 业务逻辑生成需要提供足够的上下文和约束
  4. AI辅助代码审查能发现潜在问题和优化点
  5. 保持对AI生成代码的批判性审查，确保质量和安全
